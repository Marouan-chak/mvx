# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately.

- Email: security@marouan.net (preferred)
- If email is unavailable, open a GitHub issue with minimal details and request a private channel.

Please include:
- A clear description of the issue.
- Steps to reproduce.
- Impact assessment (what could be compromised).
- Any suggested fixes or mitigations.

We will acknowledge receipt within 72 hours and work on a fix as soon as possible.
